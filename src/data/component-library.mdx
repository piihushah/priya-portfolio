# Component Library Showcase

A minimal component library demo highlighting the core building blocks in this Astro components project.

- Button component
- Card component (ImageCard)
- Section layout component
- Variants system (colors, sizes, shadows, borders)

## 4. Variants system

All components are built on a shared variants system in `utils/variants.ts`.

- Button: `type` (`primary`, `inverse`, etc.), `size` (`sm`, `md`, `lg`)
- ImageCard: `rounded`, `shadow`, `border`
- Section: `bg`, `padding`, `spacing`
